# Пятая лабораторная по "Моделирование систем"

Код - [main](main.go) и [обучение модели](r.go)

## Описание алгоритма нахождения коэффициентов регрессии

1. Создаются две матрицы X (со значениями факторов) и Y (со наблюдаемыми значениями)
2. Вычисляется транспонированная матрица X
3. Транспонированная матрица X умножается на матрицу X
4. Вычисляется обратная матрица к X\*Xt
5. Транспонированная матрица X умножается на матрицу Y
6. Полученные матрицы (X\*Xt)^-1 и Xt\*Y умножаются друг на друга
7. Результирующая матрица будет сдержать необходимые коэффициенты для каждого из факторов

## Описание алгоритма нахождения статистической значимости с помощью критерия Фишера

1. Вычисляется дисперсия для каждого наблюдаемых значений по формуле D = sum(x^2)/n - (sum(x)/n)^2
2. Вычисляется дисперсия для каждого фактора
3. Вычисляется критерий Фишера по формуле Dy/Dx
4. Значения критерия Фишера сравниваются с табличным критическим значением для данного количества наблюдаемых значений и факторов
5. Если значения критерия Фишера меньше табличного значения, то регрессия статистически значима

## Описание алгоритма нахождения коэффициента корреляции Пирсона

1. Для каждой комбинации факторов вычисляется коэффициент корреляции Пирсона по формуле r = sum(x-x_mean)\*(y-y_mean) / sqrt(sum(x-x_mean)^2) \* sqrt(sum(y-y_mean)^2)
2. Если модуль полученного коэффициент корреляции Пирсона больше 0.9 то присутствует сильная корреляция между факторами

## Описание алгоритма нахождения коэффициента детерминации

1. С помощью полученных коэффициентов регрессии вычисляется множество спрогнозированного наблюдаемых значений
2. С помощью искомого и спрогнозированного множества вычисляется коэффициент детерминации по формуле R^2 = 1 - sum((y - y_pred)^2) / sum((y - y_mean)^2)
3. Чем ближе R^2 к единице тем выше качество модели

## Описание алгоритма вычисления RMSE

1. С помощью полученных коэффициентов регрессии вычисляется множество спрогнозированного наблюдаемых значений
2. С помощью искомого и спрогнозированного множества вычисляется RMSE по формуле RMSE = sqrt(sum((y - y_pred)^2) / n)
3. Чем ближе RMSE к нулю тем выше качество модели
